# Formula Kit Development Project

This Unity project is provided for working on Formula Kit within the Unity Editor.

## Getting Started
1. Open the `DevelopmentProject` folder from Unity Hub.
2. Unity will automatically resolve the local package dependency declared in `Packages/manifest.json`.
3. Import the **Formula Builder Demo** sample from the Package Manager to explore the tooling and runtime examples.

The project intentionally contains only the configuration files required for Unity to generate the remaining project assets on first load.
